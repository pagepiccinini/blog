ANOVAs and Linear Models Part 2

Introduction

In the last post we looked at linear regressions and particularly what happens when you add an interaction to a linear regression. We found that baselines make it difficult to interpret main effects, and confirmed that plotting data is the best way to really understand whats going on. Today well look at the same data as last time, but use a statistic that gets rid of our baseline issue for interpreting main effects

TAKE AWAY POINTS FROM THIS POST

  • ANOVAs can be used to look at main effects when an interaction is in the model.

  • When running an ANOVA on participants or items it is standard practice to get means for each participant or item first.

  • ANOVAs cannot take into account variability within participant or item if the data is averaged.

Model #1: ANOVA with two variables and an interaction

To start well use the same data as in the previous post, but now running an ANOVA instead of a simple linear model. As a reminder, this is reaction times for a lexical decision experiment. Previously we looked at the effect of previous response (correct, incorrect) and sex of participant (female, male) on response times. In addition to using the languageR and ggplot2 packages, Ill also be using dplyr for some data manipulation. Well start be replotting the figure of our interaction, since you should always plot your data before analyzing it.

library(languageR)
library(ggplot2)
library(dplyr)
library(gridExtra)

To build the ANOVA well use the aov call, but other than that the syntax is the same as our lm call.

lexdec_prevcorXsex.aov = aov(RT ~ PrevCorrect * Sex, data = lexdec)
summary(lexdec_prevcorXsex.aov)
##                   Df Sum Sq Mean Sq F value  Pr(>F)   
## PrevCorrect        1   0.51  0.5103   8.826 0.00301 **
## Sex                1   0.22  0.2235   3.865 0.04946 * 
## PrevCorrect:Sex    1   0.28  0.2809   4.859 0.02764 * 
## Residuals       1655  95.69  0.0578                   
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

The ANOVA above finds a significant effect of previous response, sex, and a significant interaction of previous response and sex. Recall that for the linear model we lost our significant main effects due to the interaction adding baselines for each variable. The ANOVA recenters each variable for us, such that there is no baseline for main effects. As a result, the ANOVA gives us the results of the 2.917main effects for each variable, and the interaction.

To confirm this look at our linear models again. Youll see that the p-values for the main effects in the first model without the interaction are roughly equivalent to the p-values for the main effects in the ANOVA, and the p-value for the interaction in the second model is roughly the same as in the ANOVA. Note though that in the linear model p-values are based on t-values, while in the ANOVA they are based on F-values.

lexdec_prevcor_sex.lm = lm(RT ~ PrevCorrect + Sex, data = lexdec)

Model #2: ANOVA by participant

It looks like ANOVAs are the way to go to be able to interpret main effects and interactions. However, there is some key variability that we havent been accounting for. Each participant has multiple trials, and a given participant may behave differently from another participant. Right now though our model treats two trials from the same participant just like two trials from two unique participants. To account for this variability researchers often conduct an ANOVA by-participant (or by-subject) in which the mean of the depenent variable (here reaction times) is found for each participant for each variable in the analysis. The ANOVA is then run on these means, instead of the original raw data.

To show how this is done Ive summarized the data with means for each participant for each level of previous response (correct, incorrect). You can see the first few rows below. Note that there are two rows for each participant, since no participant was 100% correct or incorrect. Furthermore, each participant also has only one sex assigned to them, so both rows for a given participant has the same value for sex. Ive named this summary column RT_Mean.

head(lexdec_byparticipant)

Heres a boxplot of what our two variables look like, now plotting the means for each participant. Comparing to the boxplot with all the raw data, youll notice its a bit different, but not dramatically so. Particuarly there are fewer outliers the effect of previous responses seems larger for both sexes.

Now we can run our same ANOVA as from before, however with far fewer data points due to the summarizing. As a result none of our effects are significant as can be seen in the output of the ANOVA. Despite the fact that the effect of the intearction looks larger in the boxplot the reduction in power (from 1659 data points to 42 data points) makes the effect go away.

lexdec_byparticipant_prevcorXsex.aov = aov(RT_Mean ~ PrevCorrect * Sex, data = lexdec_byparticipant)
summary(lexdec_byparticipant_prevcorXsex.aov)
##                 Df Sum Sq Mean Sq F value Pr(>F)
## PrevCorrect      1 0.0470 0.04701   1.325  0.257
## Sex              1 0.0355 0.03547   1.000  0.324
## PrevCorrect:Sex  1 0.0136 0.01364   0.384  0.539
## Residuals       38 1.3486 0.03549

However, this ANOVA is still failing to account for some variance. The ANOVA above does not account for the fact that the previous response variable is within-participant, as each participant has previous responses that are correct and incorrect. To deal with this we can add an error term to our model, basically telling the ANOVA to compare the previous reponse levels within participant instead over the whole dataset. As you can see previous reponse is now signifincat in our new model.

lexdec_byparticipant_prevcorXsex.aov = aov(RT_Mean ~ PrevCorrect * Sex + Error(Subject/PrevCorrect), data = lexdec_byparticipant)
summary(lexdec_byparticipant_prevcorXsex.aov)

Model #3: ANOVA by item

With our current model we account for the variablity across participants, but we dont account for any varaibitily by items. Just as two data points from the same participant cannot be treated the same as two data points from two unique participants, two data points from the same stimulus cannot be treated the same as two data points from two unique stimuli. To account for this we can run a by-items ANOVA. Our summariziing procedure is the same as for the by-participants ANOVA, only now well get means for each item, in this case the word presented to participants.

As you can see above, this time we get two means for each sex for a given word. However, previous reponse is a little more complicated, as for some words there both previous responses occured (correc, incorrect), but for some words only one. For example for the word almond there are four rows (correct - F, correct - M, incorrect - F, incorrect - M), but for woodpecker there are only three rows (correct - F, correct - M, incorrect - F).

So, sex is a within-item varaible and previous response is at least not systmeatically a within-item variable. If you are ever in doubt about whether a variable is within or between participant or item an easy trick is to run an xtabs call like the ones below. If you see any 0s then you cant call it a within-participant or -item variable. For example below we see that there are always 14 females and 7 males for every word, but there are sometimes 0s for incorrect previous responses.

head(xtabs(~Word+Sex, lexdec))

## Word         F M

##   ant       14 7
##   apple     14 7
##   apricot   14 7
##   asparagus 14 7

Again we can plot our new summarized data against the original raw data. Again, the summarized data looks different from the raw data.

We can now run our same ANOVA as before but our error term will be for item instead of participant. Also, well only tell the model that sex is a within-item variable since previous response is not systematically within-item. This output is a little more complicated than the by-participant output.

lexdec_byitem_prevcorXsex.aov = aov(RT_Mean ~  Sex + Error(Word/Sex), data = lexdec_byitem)
summary(lexdec_byitem_prevcorXsex.aov)
